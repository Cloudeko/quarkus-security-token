== Installation

If you want to use this extension, you need to add the `io.quarkiverse.quarkus-security-token:quarkus-security-token` extension first to your build file.

For instance, with Maven, add the following dependency to your POM file:

[source,xml,subs=attributes+]
----
<dependency>
    <groupId>io.quarkiverse.quarkus-security-token</groupId>
    <artifactId>quarkus-security-token</artifactId>
    <version>{project-version}</version>
</dependency>
----
